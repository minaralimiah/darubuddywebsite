What is Darubuddy?

Darubuddy is a lightweight web application inspired by minimal, modern design. It allows you to calculate how much each person owes for shared food and drinks. No more complicated math or awkward conversations at the end of a fun evening—Darubuddy handles the numbers for you.

  • Simple interface: add your friends and enter the total costs—you’ll see each share instantly.
  • Store sessions: every calculation is saved locally so you can review previous splits any time.
  • Responsive design: the layout adapts gracefully to mobile, tablet and desktop screens.
